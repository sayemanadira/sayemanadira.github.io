Real-Time Time Scale Modification

Spring 2025 Digital Signal Processing Partner(s): Jared Carreño Python
Numpy Librosa Matplotlib PyAudio Real-time Systems

Overview

Developed a real-time implementation of real-time time scale modification (TSM) algorithms that allow users to change the speed of audio playback without affecting pitch. This is crucial for applications like live audio mixing and other interactive audio tools.

Technical Details

The system combines three core techniques: Overlap-Add (OLA) for preserving percussive transients, Phase Vocoder (PV) for maintaining harmonic coherence, and Harmonic-Percussive Separation (HPS) to split the input into components best suited for each algorithm. By processing harmonic elements with PV and percussive elements with OLA, then recombining them, the system produces high-fidelity time-stretched audio in real time.

Results

Our model achieved real-time shown in our runtime analysis. To meet real-time constraints, the design incorporates an efficient buffering system and dynamic adaptation to user-defined time-stretch factors. HPS is performed offline to reduce computational load, while synchronized processing and careful hop-size alignment ensure smooth playback. Runtime analysis shows that the implementation consistently operates well below real-time thresholds across various input lengths and stretch factors, confirming its suitability for real-world use. This work demonstrates a scalable and practical hybrid approach to real-time audio transformation.
